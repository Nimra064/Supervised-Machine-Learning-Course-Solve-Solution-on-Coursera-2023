# 📘 Supervised Machine Learning Course Solutions (Coursera 2023)

This repository contains my solutions and notes for the **Supervised Machine Learning: Regression and Classification** course from the Machine Learning Specialization by Andrew Ng on Coursera.

## 📌 About the Course

This course focuses on building foundational machine learning models including:

- Linear Regression
- Logistic Regression
- Classification Algorithms
- Model Evaluation Techniques
- Feature Engineering
- Gradient Descent Optimization

It is part of the **Machine Learning Specialization** by Stanford University & DeepLearning.AI.
