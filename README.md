# 📘 Supervised Machine Learning Course Solutions (Coursera 2023)

This repository contains my solutions and notes for the **Supervised Machine Learning: Regression and Classification** course from the Machine Learning Specialization by Andrew Ng on Coursera.

---

## 📌 About the Course

This course focuses on building foundational machine learning models, including:

- Linear Regression
- Logistic Regression
- Classification Algorithms
- Model Evaluation Techniques
- Feature Engineering
- Gradient Descent Optimization

It is part of the **Machine Learning Specialisation ** by Stanford University & DeepLearning.AI.

---

## 🛠️ Technologies Used

- Python 🐍
- NumPy
- Pandas
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## 🚀 How to Run the Project

```bash
# 1. Clone the repository
git clone https://github.com/Nimra064/Supervised-Machine-Learning-Course-Solve-Solution-on-Coursera-2023.git

# 2. Navigate to the project directory
cd Supervised-Machine-Learning-Course-Solve-Solution-on-Coursera-2023

# 3. Install required dependencies
pip install -r requirements.txt

# 4. Launch Jupyter Notebook
jupyter notebook
```
